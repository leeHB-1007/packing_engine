주문 텍스트를 넣고 바로 패킹 결과를 확인하세요.

빠른 시작

  1. `uvicorn packing_api:app --reload` 실행
  2. 브라우저에서 `http://127.0.0.1:8000/app` 접속
  3. 주문 입력 후 결과 확인

주문 입력

입력 형식 예시

  • `상품명 / 수량`
  • `상품명 수량`
  • 앞에 `완박스`, `재포장`, `패킹리스트 yes/no` 줄이 있어도 됩니다.
